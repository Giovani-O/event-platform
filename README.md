### NPM Commands  

Install node_modules  
``` npm i ```  

Run in development mode  
``` npm run dev ```  

You will need to set some local variables in a ``` .env.local ``` file  
``` VITE_API_URL ``` & ``` VITE_API_ACCESS_TOKEN ```
